## Naming Convention
- `.py` - Python scripts (cross-platform)
- `_unix.sh` - Shell scripts for Unix/Linux systems
- `_mac.sh` - Shell scripts for macOS only
- `_win.*` - Windows scripts (*.ps1 for PowerShell, *.bat for Batch)
